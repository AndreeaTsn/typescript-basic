14 - Verifica daca un numar este prim

Cerinta: un numar este prim daca este mai mare sau egal cu 2 si NU se imparte exact la niciun numar intre 2 si numar - 1. Exemple: 2, 3, 5, 7, 11 sunt prime. 4, 6, 9 nu.

In script.ts, declara numar (number). Cu o bucla si un flag boolean (estePrim) decide. Afiseaza un text de forma "7 este prim: true" sau "9 este prim: false". Sugestie: foloseste break imediat ce gasesti un divizor.
