17 - Repeta un caracter de n ori

Cerinta: construieste un string in care un caracter ales este repetat de n ori. Exemplu: pentru "*" de 5 ori rezulta "*****". NU folosi "*".repeat(5).

In script.ts, declara caracter (string) si deCateOri (number). Cu o bucla for construieste rezultatul. Afiseaza un text de forma "* x 5 = *****".
