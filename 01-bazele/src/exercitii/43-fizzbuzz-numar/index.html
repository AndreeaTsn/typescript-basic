43 - FizzBuzz pentru un singur numar

Cerinta: pentru un numar dat afiseaza:
  • "FizzBuzz" daca este divizibil cu 3 si cu 5,
  • "Fizz" daca este divizibil doar cu 3,
  • "Buzz" daca este divizibil doar cu 5,
  • numarul ca text, in rest.

In script.ts, declara numar de tip number. Atentie la ORDINEA conditiilor: cazul cel mai specific (cu 3 si cu 5) trebuie verificat PRIMUL. Testeaza cu 15, 9, 10, 7.
