42 - Categorie temperatura (mai multe ramuri)

Cerinta: dupa temperatura exterioara, afiseaza categoria:
  • temperatura < 0 → "frig"
  • 0 - 14 → "racoare"
  • 15 - 24 → "placut"
  • 25 - 34 → "cald"
  • >= 35 → "canicula"

In script.ts, declara temperatura (number). Cu if/else if/else afiseaza un text de forma "La 18°C: placut". Testeaza cu -5, 10, 20, 30, 38.
