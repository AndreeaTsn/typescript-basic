30 - Tipul unui triunghi

Cerinta: avand trei laturi ale unui triunghi, afiseaza ce tip este.
  • toate trei egale -> "echilateral"
  • exact doua egale -> "isoscel"
  • toate diferite -> "oarecare"

In script.ts, declara trei variabile numerice a, b, c. Afiseaza in cutia verde un text de forma "Triunghi (5, 5, 8) este isoscel".
