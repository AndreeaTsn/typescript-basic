09 - Categorii de varsta

Ce inveti: cum verifici daca o valoare se afla intr-un INTERVAL, folosind operatorul && (de ex. varsta >= 13 && varsta <= 17).

Programul iti spune categoria pe baza varstei:

  • sub 13 ani => copil
  • 13 - 17 ani => adolescent
  • 18 - 64 ani => adult
  • 65+ ani => senior
