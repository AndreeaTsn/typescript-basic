28 - Divizibil cu 3 si cu 5

Cerinta: verifica daca un numar este divizibil SIMULTAN cu 3 si cu 5.

In script.ts, declara numar (numar intreg) si stabileste un boolean care arata daca este divizibil cu 3 si cu 5 in acelasi timp. Afiseaza in cutia verde un text de forma "15 este divizibil cu 3 si cu 5: true".
