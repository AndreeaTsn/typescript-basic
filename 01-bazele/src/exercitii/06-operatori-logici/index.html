06 - Operatori logici

Ce inveti: operatorii logici, folositi pentru a combina valori boolean:
  • && SI - true doar daca AMBELE sunt true
  • || SAU - true daca MACAR UNA este true
  • ! NU - inverseaza (true devine false si invers)

Scenariu: vrem sa stim daca o persoana poate conduce. Conditii: sa fie majora si sa aiba carnet.
