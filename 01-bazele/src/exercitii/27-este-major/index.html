27 - Este major (boolean dintr-o comparare)

Cerinta: stabileste daca o persoana este majora pe baza varstei. Salveaza rezultatul intr-un boolean si afiseaza-l.

In script.ts, declara varsta (numar) si o variabila boolean esteMajor obtinuta dintr-o comparare (fara if). Afiseaza in cutia verde un text de forma "Are 17 ani. Este major: false".
