08 - Calificativ pe baza notei

Ce inveti: mai multe ramuri de else if in lant. Cum verifici un interval de valori si afisezi un calificativ pe baza unei note.

Schimba valoarea notei in script si reincarca pagina. Pentru note intre 1 si 10, programul afiseaza calificativul corespunzator.

  • nota >= 9 => Excelent
  • nota >= 7 => Bine
  • nota >= 5 => Suficient
  • altfel => Insuficient
