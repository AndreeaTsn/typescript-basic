34 - Indicele de masa corporala (BMI)

Cerinta: calculeaza BMI si stabileste categoria. Formula: BMI = greutate / (inaltime * inaltime), unde greutatea este in kg si inaltimea in metri.
  • BMI < 18.5 -> "subponderal"
  • 18.5 - 24.9 -> "normal"
  • 25 - 29.9 -> "supraponderal"
  • 30+ -> "obez"

In script.ts, declara greutate (kg) si inaltime (m). Afiseaza in cutia verde un text de forma "BMI = 22.86 (normal)". BMI rotunjit la doua zecimale.
